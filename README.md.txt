# Market & Delivery System

## Description
A web-based application for managing a market and enabling online product ordering and delivery.

## Features
- Product listing
- Order creation
- Daily offers
- Invoice system
- Role-based system

## Architecture
Layered architecture:
- Models
- Services
- Data
- UI

## Technologies (Planned)
- Node.js (Express)
- EJS (Frontend)
- PostgreSQL (future)
- CSV (current phase)
